Basic Usage

The Simplest Use Case

Simply place Popeye inside the element you wish to anchor it to. In the example below, we're placing Popeye inside the button that causes it to open.

Popeye can wrap any element: in this case, a <div className="dropdown">.

click me
<a className="button button-main" onClick={ this.toggleDropdown }> click me <Popeye opened={ this.state.popover1 } onToggle={ this.toggleDropdown }> <div className="dropdown"> Aye aye, Captain! </div> </Popeye> </a>
